# Erhalten von spkg

Du kannst spkg auf unserem GitHub herunterladen

# Grundlegende Verwendung

Die Verwendung von spkg ist relativ einfach, du kannst einen Befehl aufrufen und Argumente hinzufügen, wie zum Beispiel:

<Tip>Du kannst auch spkg's eingebaute Hilfe verwenden, indem du `spkg help` ausführst.</Tip>

```bash
spkg install gba-driver
```

# Installieren eines Paketes

Du kannst ein Paket installieren, indem du folgenden Befehl verwendest:

```bash
spkg install [Paketgruppe]/<Paketname>
```

`Paketgruppe` ist Optional, wenn es mehrere Pakete mit demselben Namen geben soll, verwende bitte `Paketgruppe`

Ersetze `<packagename>` mit dem Paket welches du installieren willst, z.B. `htop`, `libbismuth-git` oder `python3.12`.

# Deinstallieren eines Paketes

Das Deinstallieren eines Paketes ist ziemlich identisch zum Installieren, führe einfach den selben Befehl aus den du zum installieren verwendet hast,
ersetze aber `install` mit `uninstall`, Beispielsweise:

```bash
spkg remove [Paketgruppe]/<Paketname>
```

Paketgruppen gelten auch für das Entfernen von Paketen.
